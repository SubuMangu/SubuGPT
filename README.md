# SubuGPT
## Overview
This project is an ongoing effort to build a simplified Large Language Model (LLM) from scratch using PyTorch. It implements a basic self-attention mechanism and a rudimentary text generation loop, serving as a foundation for understanding transformer-based architectures like GPT. The goal is to explore the core concepts of LLMs while progressively expanding the implementation.

**Status**: Work in progress.

## Features
- **SimpleSelfAttention**: A minimal self-attention module, the core building block of transformer models.
- **Text Generation**: A basic autoregressive generation loop to produce token sequences.
- Built with PyTorch for flexibility and learning purposes.
